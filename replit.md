# Abdouni Douae - Portfolio

## Overview
A personal portfolio website for Abdouni Douae, a Computer Engineering student. The site showcases projects, skills, certifications, and contact information with an animated network background and smooth scroll animations.

## Project Architecture
- **Type**: Static HTML/CSS/JS website (no build system, no frameworks)
- **Entry point**: `index.html`
- **Styles**: `CSS/index.css`
- **Scripts**: `script.js` (scroll reveal, typing effect, sidebar), `animated-background.js` (canvas network animation)
- **Assets**: `IMAGES/` directory containing profile photo, project screenshots, certificates, and CV PDFs

## How to Run
- Development: `python server.py` serves the site on port 5000 with cache-control headers
- Deployment: Configured as a static site serving the root directory
